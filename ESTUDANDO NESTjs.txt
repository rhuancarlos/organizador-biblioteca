ESTUDANDO NESTjs


1- Instala o nestjs + criar o projeto
  $ npm i -g @nestjs/cli
  $ nest new project-name
  respectivamente

2- Acessar pasta do projeto e remover por hora arquivos desnecessarios pois irei criar os arquivos de que preciso.
2.1 - pasta 'src' deixar apenas a main.ts e 'app.module.ts'.
2.2 - no 'app.module.ts' remover as linhas que fazem o importe dos arquivos removidos.

3- Acessar pasta 'src' source

3.1 - Criar pasta Controllers (via terminal: nest g controller <Nome_do_controller>)

Obs:... toda vez que criar um novo controler no projeto tem que ir no 'app.module.ts' e inserir na chave de 'controller'
o controller recém criado, pois só assim aplicação entederá que este controller faz parte do processo.
